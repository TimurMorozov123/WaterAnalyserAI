# WaterAnalyserAI

WaterAnalyserAI este o aplicație cu inteligență artificială care analizează imagini cu apă pentru a detecta alge dăunătoare, microplastice și bacterii, verificând rapid siguranța apei.

**Descriere**

Water AI Multi Detector este o aplicație cu inteligență artificială care analizează imagini cu probe de apă pentru a detecta alge dăunătoare, microplastice și bacterii. Programul ajută utilizatorul să verifice rapid calitatea apei și siguranța utilizării acesteia.

**Funcționalități:**

1. Detectează alge și protozoare din apă (EMDS7)
2. Identifică și numără microplastice
3. Clasifică bacterii și semnalează speciile periculoase
4. Afișează rezultate direct pe imagine (cu dreptunghiuri de detecție)
5. Oferă verdict de siguranță a apei (sigură/nesigură)
6. Generează recomandări de utilizare (agricultură, animale, uz casnic, acvacultură, industrie)
7. Permite selectarea unuia sau mai multor modele AI

**Cum utilizați programul nostru:**

1. Instalează bibliotecile necesare în terminal:
   - pip install tkinter ultralyics pillow
2. Rulează fișierul principal:
   - python app.py

**Utilizare în aplicație:**

1. Apasă *Select Image* și alege o imagine
2. Bifează modelele dorite (EMDS7 (alge și protozoare), Microplastic, Bacteria)
3. Apasă *Load Models*
4. Apasă *Run Detection*
5. Vezi rezultatele și verdictul apei în fereastră
